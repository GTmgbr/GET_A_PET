# 🐾 Get A Pet

Sistema web completo para adoção de pets que permite cadastro de usuários, autenticação, cadastro de pets, upload de imagens, edição/exclusão de pets e solicitação de adoção.

# Tecnologias Utilizadas

## Backend
Node.js, Express, MongoDB, Mongoose, JWT Authentication, Multer, MVC, RESTful API
![Node.js](https://img.shields.io/badge/Node.js-339933?style=for-the-badge&logo=nodedotjs&logoColor=white)

## Frontend
React, React Router DOM, Axios, Context API, CSS Modules

# Funcionalidades

## Usuários
- Cadastro de usuário
- Login/autenticação com JWT
- Atualização de perfil
- Upload de foto de perfil

## Pets
- Cadastro de pets
- Upload de múltiplas imagens
- Edição de pets
- Exclusão de pets
- Visualização detalhada
- Sistema de adoção
- Conclusão de adoção

## Segurança
- Rotas protegidas
- Verificação de token JWT
- Controle de permissões
- Upload validado de imagens
